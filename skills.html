Bash

I use Linux on my personal computer and am comfortable using the command line for file and system operations. I first used Bash scripts to automate tasks and solve problems as part a Big Data Programming module. I now use bash scripts to automate and simplify certain process on my personal computer. One example of this is the following batch script that is triggered to run every time I open my budget spreadsheet via a macro in the spreadsheet. It obtains the current price of an index fund I am interested in, which is then automatically inserted into a particular cell of the spreadsheet. 

			#!/bin/bash
curl -s -A "Mozilla/5.0" \
  "https://query1.finance.yahoo.com/v8/finance/chart/VWCE.DE?range=5d&interval=1d" \
  | grep -Po '"regularMarketPrice": *\K[0-9.]+' \
  > ~/Documents/Money/price.txt
